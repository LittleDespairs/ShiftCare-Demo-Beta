Weekly schedule table from Sunday to Saturday

Home Employees Weekly Preferences API Docs
Select a week and a position, then click Load Schedule

How it works

  • The table shows only employees assigned to the selected position.
  • Each cell has 3 checkboxes: M = morning, E = evening, N = night.
  • Changes are saved immediately after each click.
  • Backend rules still apply, including conflicts and weekly off-day requests.
