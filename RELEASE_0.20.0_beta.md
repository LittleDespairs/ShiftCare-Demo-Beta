# ShiftCare 0.20.0 Beta

## Release Focus

This release focuses on faster work with large employee rosters and a clearer weekly schedule workspace.

## Changed

- Added search and filtering across staff-heavy pages.
- Added position filters on Employees and Weekly Preferences.
- Reworked main-app Weekly Preferences into an all-employee table, so requests can be reviewed without preselecting employees.
- Refreshed the Schedule page with a denser weekly table, persistent coverage status above the board, retained shift/status creation actions, and a right-side operational panel.
- Added a more polished week picker on weekly views.
- Updated main, demo, and release metadata to `0.20.0_beta`.
